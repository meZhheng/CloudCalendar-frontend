## 运行步骤：
### 配置环境
确认本机环境安装有node和npm
```
node -v     #检查是否安装node
npm -v      #检查是否安装npm
```
建议版本
```
node==18.12.1
npm==8.19.2
```
### 安装依赖
在项目主目录下运行：
```
npm install
```
